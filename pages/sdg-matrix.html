Matriu ODS x sistemes

Aquesta vista resumeix, en una sola matriu, com es distribueixen els hits detectats per text2sdg per a cada sistema i cada ODS. Pots moure't entre l'abast de centre, ensenyament o assignatura concreta sense canviar de pàgina.

Les cel·les mostren el nombre de deteccions (n_hits) per parella sistema-ODS. El color s'ajusta a l'escala relativa de la selecció activa per ajudar a localitzar de seguida on es concentra més intensitat.

Els colors es recalculen per a cada selecció visible. Això facilita comparar patrons interns dins de la mateixa matriu, però no equival a una escala absoluta comuna entre centres o ensenyaments diferents. Les cel·les grises indiquen 0 hits.
